# MyConti


Software per gestire il budget personale e fare previsione di spesa per ora (Credem)


### How-To 

1. Inserire i report .csv della carta prepagata in carta/anno
2. Inserire i report .csv del conto corrente in files/anno
3. Inserire i report .csv di  paypal in paypal/anno

####  Le categorie di spesa possono essere ridefinite sulla base personale
